### K-factor и рост

```
┌─────────────────────────────────────────────────────────────┐
│  K = (пригласившие / всего) × среднее приглашённых          │
│                                                             │
│  K < 1.0 → рост замедляется                                 │
│  K = 1.0 → стабильность                                     │
│  K > 1.2 → экспоненциальный рост                            │
│                                                             │
│  Пример: 100 юзеров, 30 пригласили по 2 человека            │
│  K = (30/100) × 2 = 0.6 → нужно улучшать!                   │
└─────────────────────────────────────────────────────────────┘
```
