### 4.3 Как генерировать ссылки для клиентов

**Формат VLESS+Reality ссылки:**

```
vless://UUID@SERVER:PORT?encryption=none&flow=xtls-rprx-vision&security=reality&sni=www.google.com&fp=chrome&pbk=PUBLIC_KEY&sid=SHORT_ID&type=tcp#SERVER_NAME
```

**Пример:**
```
vless://abc123@nl.example.com:443?encryption=none&flow=xtls-rprx-vision&security=reality&sni=www.google.com&fp=chrome&pbk=XYZ123&sid=abc456&type=tcp#Netherlands
```

**Go код для генерации:**

```go
func GenerateVLESSLink(cfg VLESSConfig) string {
    params := url.Values{}
    params.Set("encryption", "none")
    params.Set("flow", cfg.Flow)
    params.Set("security", "reality")
    params.Set("sni", cfg.SNI)
    params.Set("fp", cfg.Fingerprint)
    params.Set("pbk", cfg.PublicKey)
    params.Set("sid", cfg.ShortID)
    params.Set("type", "tcp")
    
    link := fmt.Sprintf(
        "vless://%s@%s:%d?%s#%s",
        cfg.UUID,
        cfg.Server,
        cfg.Port,
        params.Encode(),
        url.QueryEscape(cfg.Name),
    )
    
    return link
}
```
