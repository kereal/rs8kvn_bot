# Paperclip v2.3.1

> Released: 2026-07-08

## Highlights

- **Clash YAML Support**: Added native support for Clash-formatted subscription sources.
- **Improved Performance**: Implemented bounded concurrency for fetching upstream subscription nodes to prevent resource exhaustion.
- **Enhanced Reliability**:
  - Improved error handling and log noise reduction.
  - Added concurrent-safe tests for subscription aggregation.
  - Corrected retry counter logic for subscription syncing.

## Improvements

- Refactored bot and subserver codebases to remove unused health checks and legacy abstractions.
- Hardened internal type conversions.
- Optimized concurrent fetching mechanisms for better reliability under load.

## Fixes

- Fixed potential data races in tests when sharing mock objects across parallel subtests.
- Fixed header preservation and configuration parsing issues in share links.
- Corrected retry count increment order during synchronization.
- Fixed issue where empty usernames caused log noise; now correctly captures `chat_id`.
- Ensured proper fallback for usernames in subscriptions.
