# Roadmap — rs8kvn_bot

**Версия:** v2.3.0  
**Обновлено:** 2026-06-08

## Текущий статус
- **Ветка:** `plan-mechanics`
- **Пользователи:** ~10 клиентов
- **Покрытие тестами:** ~85% (см. `test-info` для деталей)
- **Архитектура:** plan-based, nodes/plan_nodes, products/orders, subscription_nodes, SQLite
- **Сборка:** `go build ./...` ✅, `go vet ./...` ✅, `golangci-lint` ✅
- **Приоритет:** стабильность flows (миграция завершена в v2.3.0)

## Ближайшие приоритеты (1-2 мес)

### Цель: 50-100 платящих клиентов

### Фаза 1: Монетизация (1-2 недели) — ВЫСШИЙ
1. **Интеграция платежей** (ЮKassa / Т-Банк Pay) — 4-6 ч
   - Авто-продление после оплаты
   - Несколько планов (1/3/12 мес)
2. **Система промокодов** — 2-3 ч
   - Админ: `/promo КОД 30% 2026-12-31`
3. **Уведомления о лимите трафика** — 2-3 ч
   - Алерт при 80%, 90%, 100%
   - Cron каждый час, флаг в БД

### Фаза 2: Фичи для роста (3-4 недели)
4. **Система наград за рефералов** — 3-4 ч
   - Уровни: Бронза → Серебро → Золото → Платина
5. **Еженедельный отчёт админу** — 1-2 ч
6. **Авто-напоминания неактивным** — 1-2 ч

### Фаза 3: UX (месяц 2)
7. **Генератор многосерверных подписок** 🔥 — 1-2 дня
8. **Пауза/возобновление подписки** — 2-3 ч
9. **Сбор фидбека** — 1 ч

## Бэклог

### Простые (1-3 ч, когда будет время)
- Автоответы на FAQ
- Экспорт CSV для бухгалтерии
- QR-код с реферальной ссылкой
- Поздравления с ДР + скидка

### Средние (2-5 ч, когда будет 30+ клиентов)
- Авто-балансировка между серверами
- Автоматическая миграция при блокировках
- Система тикетов/поддержки
- Резервное копирование конфигураций
- Prometheus + Grafana
- Система лояльности (кэшбэк, VIP)

### Когда будет 50+ клиентов
- Авто-выбор протокола по стране/провайдеру
- Партнёрская программа
- A/B тестирование сообщений
- Telegram Mini App
